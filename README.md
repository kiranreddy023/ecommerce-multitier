# ecommerce-multitier
# E-commerce Multi-tier Application

## Tech Stack
- Backend: Spring Boot
- Frontend: Spring Boot (Thymeleaf)
- Database: PostgreSQL
- Containerization: Docker

## How to Run

### Using Docker
docker-compose up --build

### Manual Run
1. Start PostgreSQL
2. Run backend
3. Run frontend

## API Endpoints
- GET /api/products
- POST /api/products
